## Maproom Campus Navigation App

<b>Overview:</b> "Maproom" is a navigation app tailored for new students to effortlessly locate classrooms on campus in unfamiliar surroundings. This project, a collaboration between three team members, embodies a commitment to teamwork and effective organization. Leveraging Agile methodologies, our team effectively managed time constraints and maximized productivity all while mostly working remotely. We prioritized tasks to meet target goals and utilized a range of tools for seamless collaboration. Maproom not only highlights the relevant building but also provides a detailed local view, clearly marking the specific floor and room locations. This project showcases our collective strength, organizational skills, and dedication to making a meaningful software for our target demographic.

<b>Features:</b>

- <b>Precise Location Mapping:</b> Users receive clear indications of both the building and floor, along with a highlighted marker for the specific classroom they're searching for.

## Unity AI Survival Agent
![together](https://github.com/MyutVoilim/Projects-Portfolio/assets/54462267/00130ce1-471b-4a97-af2f-f037719e5e79)

<b>Technologies</b> C# | Unity | Visual Studio

<b>Overview</b>
This project uses C# and Unity to create an AI agent that generationally improves its survivability using a basic feed-forward neutral network in conjunction with a genetic algorithm. Each agent has several stats the determine variables like speed, visions range, vision cone count, and stamina that all affect the agents ability to find and compete for food in it's environment. Several agents are be pitted against each other with the agent that lived the longest having its genetics passed on to the next with slight mutations to further hone a progressively competitive agent. Showcasing an application of machine learning and OOP principles in C# as well as tackling unfamiliar 

<b>Features</b>
- <b>Environmental and Agent Control:</b> Ability to adjust factors like stamina drainage, food availability, and agent spawn count.
- <b>Adjustable Mutation Rate:</b> Control over genetic divergence from parent to off-spring to create or dimisish variability through generations.
- <b>Variable Toggleability:</b> Toggleability of certain agent variables such a whether increased vision range or movement speed drain stamina faster.
- <b>AI Model Persistence:</b> Functionality to save and load AI models.

  ![bigger](https://github.com/MyutVoilim/Projects-Portfolio/assets/54462267/077317d9-8d95-4599-b483-f621157f8b20)

<b>Challenges</b>
- <b>Learning Unity Platform:</b> Overcoming initial development slowdowns due to unfamiliarity with Unity, eventually mastering it through documentation and functionality demos.
- <b>Fine-Tuning Reward System Design:</b> Determining what constitutes as a successful agent such as time lived over amount of food eaten as a success metric for the neural network, focusing on efficiency rather than just quantity.
- <b>Agent Ability Balancing:</b> Fine-tuning the impact of attributes like vision range and speed on stamina, to ensure a competitive environment with multiple viable survival strategies.
